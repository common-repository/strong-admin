== Description ==

This plugin allows you to skin WordPress Admin's visual interface by selecting your own colors. Here are some basic features :

*   Truly Unlimited skins, you can choose colors in color picker.
*   Cross browser compatible, works in IE 7 and above.
*   Login screen can also be skinned and logo on login screen can be replaced.


== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
